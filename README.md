# zig-machine
A simple state machine written in zig for teaching.

## Dependencys
> - Zig 

## Installation
```bash
$ git clone https://github.com/eislerMeinName/zig-machine.git
```

## Build
```bash
$ zig build-exe main.zig
```

## Run
Run prebuild:
```bash
$ ./main
```
Or run code directly:
```bash
$ zig run main.zig
```
